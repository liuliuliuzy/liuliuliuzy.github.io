一些对于Hexo的理解

关于配置文件

就我使用的Fluid主题而言，需要注意的配置文件有两个，分别是_config.yml和_config.fluid.yml。一个是基本的hexo的配置文件，另一个是fluid主题的配置文件。

yml(yaml)是一种类似于Json格式的文件类型，常被用于存储配置信息。

配置文件的作用是规定博客的标题、作者、显示内容等，如果上述两个配置文件中有相同的配置项，那么_config.fluid.yml的优先级要高于_config.yml。

什么是front-matter

front-matter是文件最上方的一块内容，用于指定该文件的一些变量，在这里面定义的一些信息，优先级又要高于_config.fluid.yml。

如何创建新内容

  • hexo new [layout] <title>:

    • 你可以使用这条命令，创建新的文章、草稿、或者一个全新的博客页面
    • [layout]默认布局为post

  • hexo new post/page title：创建一篇文章/新的页面

  • hexo new draft title：创建草稿

  • hexo publish draft title：将一篇草稿发布
